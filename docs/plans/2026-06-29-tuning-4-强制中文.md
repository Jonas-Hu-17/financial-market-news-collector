# 调优任务 4：强制中文输出

> 致执行者：当前「综合市场观点」偶尔整段英文、个别 view 也有英文泄漏（源是英文时）。本任务在提示词层强制**简体中文**输出。小改动，TDD，跑相关分目录测试（`tests/view` 等，勿全量），单独 commit。

## 问题
`src/view/view_templates.py` 的 `NEUTRAL_SYSTEM`（单条 view + summary）和 `MARKET_VIEW_SYSTEM`（整期综合观点）没强制语言，模型遇到英文源就跟着输出英文。

## 改法

**1. `NEUTRAL_SYSTEM` 末尾加硬约束：**
```
LANGUAGE: Always write the "summary" and "view" in Simplified Chinese (简体中文),
regardless of the source article's language. Translate as needed. Keep proper nouns
(company names, tickers) as-is.
```

**2. `MARKET_VIEW_SYSTEM` 末尾加同样约束：**
```
LANGUAGE: Write the overview in Simplified Chinese (简体中文), regardless of the
source items' language.
```

**3. 保持其余不变**：中性、无投资建议、JSON 输出结构都不动。

## 测试 `tests/view/test_chinese_output.py`
- 用假 AI 客户端返回中文，验证流程正常（主要确认 prompt 含中文强制语句、不破坏解析）。
- 断言 `NEUTRAL_SYSTEM` 和 `MARKET_VIEW_SYSTEM` 文本里包含 "简体中文" / "Simplified Chinese" 字样。

## 验证
改完跑一次 `uv run horizon --marketnews --hours 24`，确认综合市场观点与各条 view 均为中文。
（注意：重跑前若 `docs/<date>.md` 是旧的，先删掉再跑；确保测试不污染真实 `data/`、`docs/`。）
